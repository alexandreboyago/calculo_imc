# Projeto para Calculo de IMC - Indice de Massa Corporea
Projeto criado em React JS

## Para utilizá-lo:
Faça o clone do diretório, a seguir abra o promp de comandos e digite:
### npm install

## Para Executrar
no promp de comandos 

### npm start
